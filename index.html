Pokemon Game

Visit

This code creates a simple browser-based game called "Gotta Catch 'em All!" where the player clicks on grass clumps to reveal hidden images representing Pokémon or Pokeballs. Each grass clump may reveal a Pokémon that the player catches, two free Pokeballs, or nothing. The game keeps track of the player's remaining Pokeballs, the number of Pokémon caught, and displays these stats along with the game instructions. It also keeps a record of recent events and allows the player to clear this history.

Memory Matching Madness

Visit

This HTML document creates a memory matching game called "Memory Matching Madness." Players select a difficulty level (easy, medium, or hard) and click "Start" to begin. The game involves flipping tiles to find matching pairs of Pokémon images. Upon completing the game, the player's time is displayed, and their score is stored in a leaderboard for each difficulty level. Players can also choose to play again by clicking the "Play Again" button.

Zelda Chat Rooms

Visit

This PHP-enabled HTML document sets up a chatroom interface. Users can select a nickname, choose a chatroom, and start chatting. The interface includes options to change the username, select different chatrooms, and exit the chat. Messages are sent and received asynchronously using AJAX calls to PHP scripts that handle message storage and retrieval. The chat history is continuously updated to display the latest messages, and users can scroll through previous messages. Additionally, there's an admin login button for administrative purposes.
